## AI-Powered Technical Product Recommendation Engine

Author: Sannidhya Gupta
Project Type: Machine Learning / Backend AI System
Tech Stack: Python, Pandas, TF-IDF, Cosine Similarity, FastAPI, Node.js, Express, React

## Project Overview

This project implements an AI-powered recommendation engine designed to help users discover and understand technical consumer products using natural language queries. The system analyzes product specifications and generates intelligent recommendations based on similarity ranking and user intent.

The solution focuses on improving consumer experience by simplifying complex technical specifications and providing explainable recommendations through machine learning and language models.

## Core Features

• Natural language product search
• Machine learning–based recommendation engine
• Specification similarity ranking using TF-IDF and cosine similarity
• Budget-aware filtering
• Explainable recommendations generated using an LLM
• REST API backend for easy frontend integration
• AMD hardware prioritization support

## Machine Learning Engine

The ML engine processes structured product specification data and converts it into a searchable feature space. The workflow includes:

1. Data preprocessing using Pandas
2. Feature combination from product specifications
3. Text normalization and cleaning
4. TF-IDF vectorization of specification features
5. Cosine similarity scoring between user query and products
6. Ranking and filtering of results

This allows the system to recommend products that best match user intent even when the query is written in natural language.

## API Integration

The ML engine is exposed through REST API endpoints using FastAPI. The frontend sends user queries to the backend, which processes the request and returns ranked product recommendations along with simplified explanations.

Example API Flow:

POST /recommend

Input:
{
"query": "laptop for coding under 70000"
}

Output:
{
"model": "...",
"price": "...",
"ram": "...",
"storage": "...",
"os": "...",
"reason": "Explanation generated by LLM"
}
